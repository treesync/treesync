# treesync

Sync your whole code folder across devices & agents — full-fidelity, end-to-end encrypted.

## Install

```sh
curl -fsSL https://treesync.dev/install.sh | sh
```

Then sync the folder that holds your repos:

```sh
cd ~/code && treesync init
```

This repository hosts the released `treesync` CLI binaries. Learn more at https://treesync.dev.
